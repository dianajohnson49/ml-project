## Dataset
Primary dataset: [Gaming and Mental Health](https://www.kaggle.com/datasets/shaistashahid/gaming-and-mental-health)

## Gaming and mental health dataset information
27 features, 1000 datapoints     
10 Categorical features (including target)  
4 Binary categorical features  
1 Unique numerical feature (record_id)  
12 Numerical features (non-unique)  

Four binary features all use TRUE/FALSE for 1/0   
Categorical features in order: Gender, Game genre, Primary game, gaming platform, sleep quality, sleep disruptions, academic performance, mood state, mood swings, gaming addiction risk level (target)  
Most features showed distributions that should be suitable for classification 

## Scripts 
Tasks were broken up into various script files. To run any given script, open the file in an IDE software, download
the necessary packages, and click run. Most scripts require data file imports in the structure demonstrated in the Git repo.
